High-level API

Horned Sungem High Level API

Version 0.1.0

HS.__init__()

  • Version: 0.1.0

  • Overview: initialisation

  • Syntax:

hs.HS(modelName, **kwargs)

  • Parameters:

    • modelName: built-in models’ name

    • **kwargs key arguments: (Optional)

      • ‘verbose’ Detail of log. From 0 to 2: no log, key log, full log. Default is 2

      • ‘mean’ Mean value in preprocessing. Default is 1.0

      • ‘scale’ Scale value in preprocessing. Default is 0.007843 (1⁄127.5)

      • ‘graphFolder’ Model folder path. Default is ‘../graphs/’

      • ‘zoom’ If True the buil-in cam has resolution 640x360, else 1920x1080. Default is True.

      • ‘threshSSD’ Threshold for SSD detector, range from 0-1. Default is 0.55.

  • Return: Return the HS Object instance

  • Example:

net = hs.HS('FaceDetector', zoom = True, verbose = 2)

HS.run()

  • Version: 0.1.0

  • Overview: run HS CNN once

  • Syntax:

result = net.run(img, **kwargs)

  • Parameters:

    • img: In HS-mode (without webcam), the input should be None; in webcam mode, input should be UINT8 image, can be grey image or BGR(OpenCV video capture).

  • Return: returns [Image, Output], video frame (UINT8) and inference result.

  • Example:

result = net.run(img)

HS.getParam()

  • Version: 0.1.0

  • Overview: get model parameters

  • Syntax:

modelParam = getParam(modelName):

  • Parameters:

    • modelName: model name, e.g. ‘mnist’.

  • Return: returns model parameters, e.g. [‘graph_mnist’, 0.007843, 1.0, True, (28,28), 1] correspond to Model’s name, Scale, Mean, isGrey, input size of net, model ID

  • Example:

modelParam = getParam('mnist'):

SSD-related high-level functions

HS.getBoundingBoxFromSSDResult()

  • Version: 0.1.0

  • Overview: get bounding boxes and other infos from CNN result

  • Syntax:

boxes = getBoundingBoxFromSSDResult(out_HS, size):

  • Parameters:

    • out_HS: processed result
    • size: network input size

  • Return: returns bounding box array, e.g. [[0, 0.99805, 334, 2, 595, 289]], correspond to:

    • class: face detector is fixed to 0, object detector between 0-19 according to the label (please refer to VOC dataset)
    • confidence: ranged from 0-1, the higher means the detection is more reliable
    • bounding box: 4 integers correspond to the positions of left-top corner and right-bottom corner

  • Example:

output = self.getBoundingBoxFromSSDResult(output, (300,300))

HS.plotSSD()

  • Version: 0.1.0

  • Overview: Plot bounding boxes from SSD result on the image

  • Syntax:

img = net.plotSSD(result)

  • Parameters:

    • result: the returned result from HS.run()

  • Return: returns plot image of bounding boxes

Scene-Recorder-related functions

HS.init_recorder()

  • Version: 0.1.0

  • Overview: initialise scene recorder. Please note that this is autometically called in record function

  • Syntax:

net.init_recorder(feat_dim)

  • Parameters:

    • feat_dim: feature dimension, 1024 for the built-in model

  • Example:

self.init_recorder(result[1].shape[0])

HS.record()

  • Version: 0.1.0

  • Overview: Two modes depending on the pressed key: recording mode and recognition mode:

    • Recording mode: press a key on keyboard from [1-9] to save the current image feature to the target bin. Press [r] to process saved image features and train a simple classification model.

    • Recognition mode: not press any key will return the recognition result.

  • Syntax:

result = record(result, key, **kwargs)

  • Parameters:

    • result: image and its feature description vector
    • key: pressed key returned by OpenCV cv2.waitKey, e.g. ‘1’.

    • **kwargs key parameters (Optional)

    • ‘saveFilename’ the filename for saving the features to the local folder. Default is : ‘../misc/record.dat’.

    • ‘numBin’ number of bins. Default is 5.

    • ‘threshPerc’ a threshold for pruning the redundent image features. Default is 0.3.

    • ‘metric’ metric type. Default is ‘euclidean’. For other metric please refer to the Spotify Annoy documentation.

  • Return: None or probabilities for all bins

  • Example:

key = cv2.waitKey(5)
prob = net.record(result, key)

HS.compressFeatBin()

  • Version: 0.1.0

  • Overview: compress feature bins, remove redundent image features

  • Syntax:

net.compressFeatBin()

HS.buildANN()

  • Version: 0.1.0

  • Overview: learn an ANN model (Approximate Nearest Neighbour) from stored image features, with the Spotify ANN Python library Annoy.

  • Syntax:

net.buildANN()

HS.runANN()

  • Version: 0.1.0

  • Overview: run ANN model to get the nearest neighbor index and distance of an image feature vector

  • Syntax:

result = net.runANN(feat)

  • Parameters:

    • feat: image feature vector

  • Return: probabilities for all bins

HS.saveBinsToLocal()

  • Version: 0.1.0

  • Overview: press ’s’ to save features of all bins to a local file. Default is ../misc/record.dat

  • Syntax:

net.saveBinsToLocal()

HS.loadBinsToLocal()

  • Version: 0.1.0

  • Overview: press ‘l’ to load features to bins from a local file. Default is ../misc/record.dat

  • Syntax:

net.loadBinsToLocal()

HS.resetBins()

  • Version: 0.1.0

  • Overview: press ‘p’ to reset all bins

  • Syntax:

net.resetBins()

HS.dispBins()

  • Version: 0.1.0

  • Overview: display number of image features in every bin

  • Syntax:

net.dispBins()

Utility function

HS.getImage()

  • Version: 0.1.0

  • Overview: bypass the CNN inference in HS, only get image

  • Syntax:

image = net.getImage()
  • Return: UINT8-BGR image array

HS.msg()

  • Version: 0.1.0

  • Overview: print message in console, if Verbose >= 1

  • Syntax:

net.msg(string)
  • Parameters:
    • string: string to display

HS.msg_debug()

  • Version: 0.1.0

  • Overview: print debug message in console, if Verbose >= 2

  • Syntax:

net.msg_debug(string)
  • Parameters:
    • string: string to display

HS.softmax()

  • Version: 0.1.0

  • Overview: SoftMax function

  • Syntax:

softmax_result = net.softmax(result)
