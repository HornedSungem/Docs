  • 角蜂鸟
  • 说明书
  • Type-C 数据线

安装条件

本产品支持在Ubuntu、CentOS、Fedora、TK1、MacOS、Raspbian（树莓派）操作系统上运行。

硬件参数

  • 处理器 Interl Movidius MA245X 卷积神经网络(CNN)加速芯片
  • 运行内存 512MB
  • 数据接口 Type-C USB 3.0
  • 摄像头分辨率 1920x1080 或 640x360
  • 整体功耗 小于3W
  • 重量 小于10g
  • 产品规格 63x33x38mm

使用说明

  1. 将角蜂鸟防静电包装拆除。
  2. USB Type-C端（小端）与角蜂鸟连接。
  3. USB Type-A端（大端）与PC主机或树莓派连接。
  4. 访问我们的 GitHub 页面，根据需要安装角蜂鸟的开发工具。

硬件版本说明

A.0

A.0 版本采用 USB Type-C 接口，支持 USB 3.0 协议；摄像头模块与角蜂鸟之间，采用 CSI 接口直接连接；两侧预留了双目摄像头接口，可实现立体视觉系统，目前尚未用到。

A.1、A.2

A.1 和 A.2 版本相较于 A.0，取消了两侧预留的双目摄像头接口，安装了4颗LED指示灯，并且新增了 MicroUSB 接口，可以为角蜂鸟提供额外的供电。

4颗LED灯的含义，如图所示从上到下分别为：

  • 电源指示
  • 温度指示，温度大于80℃时自动降频
  • 神经网络运行指示
  • 神经网络输入等待指示
