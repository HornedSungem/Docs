FAQs

If you still have issues, please contact us at support@hornedsungem.org

Q: Error: No devices found
A: Please wait few seconds and try again, if doesn't work please try reconnect the ports.

Q: Error: Failed to load graph, please check file path
A: If model file paths are changed, please specify the new path in the function argument, e.g. net = hs.HS('mnist',graphFolder='../MyNewPath/graph')

Q: Error: Failed to allocate graph to device, please try to re-plug the device or Exception: hsStatus.ERROR
A: Usually caused by not closing the device properly from the last application. Please try reconnecting the device.

Q: OpenCV Error: Assertion failed (VScn::contains(scn) && VDcn::contains(dcn) && VDepth::contains(depth)) in CvtHelper... line 253
A: Please try **print(image)** to ensure the existence of the image.

Q: How to replace the model on-the-fly or load multiply models in one go.
A: Currently the device must be reset to load another model. Soon it will be supported, we are working on it. 

Q: The VMware virtual machine cannot detect the device.
A: Set virtual machine USB compatibility to `USB 3.0`.
