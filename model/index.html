Built-in Models

Currently HS includes 8 built-in models:

MNIST digit recognition:

  • Version: 0.1.0

  • Filename: graph_mnist

  • Description: MNIST digit recognition

  • Example:

    • Quick Start - Hello 2018

  • Properties:

    • Channel: 1 (Grey image)
    • Scale: 0.007843
    • Mean: 1.0
    • Image Size: [28,28]
    • Type: Recognition

Mobilenet-SSD Face Detector

  • Version: 0.1.0

  • Filename: graph_face_SSD

  • Description: Mobilenet + Single-shot detector Face Detector

  • Example:

    • Mobilenet-SSD Face Detector

  • Properties:

    • Channel: 3 (RGB image)
    • Scale: 0.007843
    • Mean: [1.0, 1.0, 1.0]
    • Image Size: [300,300]
    • Type: SSD detection

Mobilenet-SSD VOC Object Detector

  • Version: 0.1.0

  • Filename: graph_object_SSD

  • Description: Mobilenet + Single-shot detector Object Detector,trained with VOC dataset,20 objects in total

  • Example:

    • Mobilenet-SSD object detection

  • Properties:

    • Channel: 3 (RGB image)
    • Scale: 0.007843
    • Mean: [1.0, 1.0, 1.0]
    • Image Size: [300,300]
    • Type: SSD detection

SqueezeNet Image Classification Model

  • Version: 0.1.0

  • Filename: graph_sz

  • Description: SqueezeNet image classification model, trained with ImageNet, 1000 classes in total

  • Example:

    • SqueezeNet Image Recognition (Low level API tutorial)

  • Properties:

    • Channel: 3 (BGR)
    • Scale: 1
    • Mean: [104, 117, 123]
    • Image Size: [227,227]
    • Type: Recognition

GoogleNet Image Recognition Model (Descriptor)

  • Version: 0.1.0

  • Filename: graph_g

  • Description: GoogleNet image recognition model as image descriptor

  • Example:

    • Scene Recorder

  • Properties:

    • Channel: 3 (RGB image)
    • Scale: 0.007843
    • Mean: [1.0, 1.0, 1.0]
    • Image Size: [224,224]
    • Type: Image descriptor

FaceNet Face Recognition Model (Image descriptor)

  • Version: 0.1.0

  • Filename: graph_fn

  • Description: FaceNet face recognition model as image descriptor

  • Example:

    • Scene Recorder

  • Properties:

    • Channel: 3 (RGB image)
    • Scale: 0.007843
    • Mean: [1.0, 1.0, 1.0]
    • Image Size: [160,160]
    • Type: Image descriptor

SketchGraph Sketch Recognition Model

  • Version: 0.1.0

  • Filename: graph_sg

  • Description: SketchGraph sketch recognition model, trained with Google’s Quick Draw! dataset, for class list please refer to class_list.txt under /misc

  • Example:

    • Sketch Recognition (Chinese blog)

  • Properties:

    • Channel: 3 (Binarised RGB image)
    • Scale: 0.007843
    • Mean: [1.0, 1.0, 1.0]
    • Image Size: [28,28]
    • Type: Recognition

OCR Chinese Character Recognition Model （Work in progress）

  • Example:

    • OCR Chinese character recognition (work in progress)

  • Properties:

    • Channel: 3 (Binarised RGB image)
