内置模型与实例说明

目前角蜂鸟包括6个内置模型：

MNIST 数字识别模型

  • 版本 Version: 0.1.0

  • 文件名称 Filename: graph_mnist

  • 描述 Description: MNIST 数字识别模型。

  • 实例 Example:

    • MNIST 数字识别 （使用入门）

  • 属性 Properties:

    • Channel: 1 （灰度图）
    • Scale: 0.007843
    • Mean: 1.0
    • Image Size: [28,28]
    • Type: 识别

Mobilenet-SSD 人脸检测模型

  • 版本 Version: 0.1.0

  • 文件名称 Filename: graph_face_SSD

  • 描述 Description: Mobilenet + Single-shot detector 人脸检测模型。

  • 实例 Example:

    • Mobilenet-SSD 人脸检测

  • 属性 Properties:

    • Channel: 3 （RGB图）
    • Scale: 0.007843
    • Mean: [1.0, 1.0, 1.0]
    • Image Size: [300,300]
    • Type: SSD检测

Mobilenet-SSD VOC物体检测模型

  • 版本 Version: 0.1.0

  • 文件名称 Filename: graph_object_SSD

  • 描述 Description: Mobilenet + Single-shot detector 物体检测模型，VOC数据集训练，共20种物体。

  • 实例 Example:

    • Mobilenet-SSD 物体检测

  • 属性 Properties:

    • Channel: 3 （RGB图）
    • Scale: 0.007843
    • Mean: [1.0, 1.0, 1.0]
    • Image Size: [300,300]
    • Type: SSD检测

  • 文件名称 Filename: graph_sg
