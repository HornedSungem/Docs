Horned Sungem Quick Start

In this tutorial Hello 2018, we will introduce how to deploy a built-in CNN model in Python using HS, with only a few lines of code!

The goal is to identify four hand-written images of 2018. We are going to deploy a simple CNN model trained with MNIST hand-writing dataset.

  • This tutorial is based on Ubuntu 16.04

Path and files

Before running the example programs, you must download and copy required files from SungemSDK-GraphModels to the corresponding folders under the examples.

SungemSDK-GraphModels/graphs -> examples/graphs
SungemSDK-GraphModels/misc   -> examples/misc
  • graphs: folder for built-in CNN model files, modification or removal of the files may cause failure on API.
  • misc: folder for images, labels and other miscellaneous files.

Hello 2018

Under python directory, type the following commands to run Hello 2018 demo

~/SungemSDK-Python/examples/apps/Hello2018$ python3 Hello2018.py

If everything goes well, the following will be displayed:

Four hand-written images are successfully identified.

Code analysis

Loading OpenCV, Numpy and HSAPI,

import cv2, sys, numpy as np
sys.path.append('../../../')
import hsapi as hs

initializing HS, built-in models and image root folder,

net = hs.HS('mnist')
imgRoot = '../../misc/2018_mnist/%d.jpg'

load four images with cv2.imread to get the image pixel values, then load to HS device through HS.run(img). HS returns and prints the result.

print('Hello')
for n in [1,2,3,4]:
	imgname = imgRoot % n
	img = cv2.imread(imgname)
	result = net.run(img)
	print(result[1])

How to only get image from built-in camera in HS

Check GetImage.py for example.

image = net.getImage()

Note that the image is in BGR format as in OpenCV.
