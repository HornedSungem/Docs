Face detector

SSD-Mobilenet Face Detector

This chapter will introduce how to use HS to deploy SSD-Mobilenet face detector

  • Based on Ubuntu 16.04

Path and files

  • Python: SungemSDK/examples/python/FaceDetector.py
  • Model file: SungemSDK/examples/graphs/graph_face_SSD

Face Detector

Under python directory, enter following command to run face detector demo:

~/SungemSDK/examples/python$ python3 FaceDetector.py

get result:

| ======= Horned Sungem ======== |
|        Device found [0]        |
|    ../graphs/graph_face_SSD    |
|     Model loaded to Python     |
|   Model allocated to device    |
| ============================== |
* *****SSD [0]: Box values****** *
...

if any face detected:

* *****SSD [1]: Box values****** *
*         Box Name: Face         *
* 360 106 591 361 - w:231 h:255  *

N indicates the number of face deteced, the lines below indicates bounding box locations

Parameter setting

Initialisation
WEBCAM = False
net = hs.HS('FaceDetector', zoom = True, verbose = 2)

Two modes:

  • if True, use USB webcam

  • if False, use built-in HS cam

  • zoom (only for HS cam): if True, the resolution is set to 640*360, otherwise 1920*1080

  • threshSSD: detection threshold, ranged 0-1, raise threshold will allow more faces to be detected but risk for false alarm

Detection
result = net.run(img)

result returns the image and detection result, stored in result[0] and result[1] respectively

Note that when using HS cam, no image is required (img = None)

a result example:

print(result[1])
[[0, 0.99805, 334, 2, 595, 289]]
  • class: fixed to 0
  • confidence: ranged from 0-1, the higher means the detection is more reliable
  • bounding box: 4 integers correspond to the positions of left-top corner and right-bottom corner
Visualisation
img = net.plotSSD(result)
