Android Hello 2018

快速开始

下面介绍如何在Android Studio开发环境下通过角蜂鸟SDK使用卷积神经网络，实现复杂计算

步骤1：准备环境

  1. 下载最新版HornedSungemSDK jar包
  2. 请确保满足一下开发环境要求
    • 确保minSdkVersion在Android 3.1以上
    • 确保设备支持OTG
    • Android Studio 2.0或以上版本
  3. 新建project工程
  4. 请确保在使用角蜂鸟相关功能及服务前，已阅读Android API,详见API文档

步骤3：添加权限

为了保证SDK正常运行，需要在AndroidManistest.xml文件下添加下列许可：

<uses-permission android:name="android.permission.READ_EXTERNAL_STORAGE"/>
<uses-permission android:name="android.permission.WRITE_EXTERNAL_STORAGE"/>
<uses-permission android:name="android.hardware.usb.host"/>
<uses-permission android:name="android.hardware.usb.accessory"/>
<uses-feature android:name="android.hardware.usb.host"/>

  4. Hello2018Thread继承HsThread,处理业务逻辑，具体代码如下:

    public class Hello2018Thread extends HsThread {

public Hello2018Thread(Activity activity, UsbLinkVsc usbLinkVsc, Handler handler) {
    super(activity, usbLinkVsc, handler);
}

@Override
public void run() {
    super.run();
    int status = allocateGraphByAssets("graph_mnist");
    if (status != HsConnectApi.HS_OK) {
        Message message = mHandler.obtainMessage();
        message.arg1 = 1;
        message.obj = status;
        mHandler.sendMessage(message);
        return;
    }
    for (int i = 1; i < 5; i++) {
        int[] ints = new int[28 * 28];
        try {
            InputStream inputStream = mActivity.getAssets().open("hello/" + i + ".jpg");
            Bitmap bitmap = BitmapFactory.decodeStream(inputStream);
            bitmap.getPixels(ints, 0, 28, 0, 0, 28, 28);
        } catch (FileNotFoundException e) {
            e.printStackTrace();
        } catch (IOException e) {
            e.printStackTrace();
        }
        float[] float_tensor = new float[28 * 28];
        for (int j = 0; j < 28 * 28; j++) {
            float_tensor[j] = Color.red(ints[j]) * 0.007843f - 1;
        }
        int status_load = mHsApi.loadTensor(float_tensor, float_tensor.length, 0);
        if (status_load == HsConnectApi.HS_OK) {
            float[] result = mHsApi.getResult(0);
            if (result != null) {
                int max = getMaxPossible(result);
                mHandler.sendEmptyMessage(max);
            }
        }
    }
}
public int getMaxPossible(float[] arr) {
    int max = 0;
    float max_f = 0;
    for (int i = 0; i < arr.length; i++) {
        float temp = arr[i];
        if (temp > max_f) {
            max = i;
            max_f = temp;
        }
    }
    return max;
}
}
