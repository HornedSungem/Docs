安装条件

本产品支持在Ubuntu、MacOS、Raspbian（树莓派）操作系统上运行。

安装说明

  1. 将角蜂鸟防静电包装拆除。
  2. USB Type-C端（小端）与角蜂鸟连接。
  3. USB Type-A端（大端）与PC主机或树莓派连接。
  4. 安装角蜂鸟开发工具 SungemSDK。

安装 SungemSDK

  • 请访问我们的Github repo来获取最新的 SungemSDK。

  • 解压文件或在终端输入下载SDK

git clone https://github.com/HornedSungem/SungemSDK.git

  • Ubuntu 16.04 或树莓派上进入SungemSDK/installer/Ubuntu_and_Raspbian，运行

    sudo ./install.sh tuna

  • Mac 系统上进入SungemSDK/installer/OSX，运行

    ./install.sh tuna

*提示： 参数’tuna’将使用国内清华pip安装镜像，可大幅提升下载速度。如在国外或不需要国内镜像可不加参数。

固件升级

新版本更新需固件升级时，请进入API文件夹然后输入：

python3 upgrade.py -f file

file为升级补丁文件
